# 🌤️ Weather Dashboard

A modern weather dashboard web application that displays real-time weather conditions and forecasts for any location using a clean and responsive interface.

---

## 📸 Preview

![Weather Dashboard](./screenshots/homepage.png)

---

## ✨ Features

- 🌍 Search weather by city
- 🌡️ Real-time temperature updates
- 🌧️ Weather condition display (rain, sun, clouds, etc.)
- 📅 7-day forecast view
- 📱 Fully responsive design
- ⚡ Fast and simple UI
- 🎨 Clean modern layout

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Weather API (e.g. OpenWeatherMap)
